# Python Flavours & Versions

## Flavours of Python
- **CPython** → Standard Python implementation (default)  
- **Jython** → Python for Java  
- **IronPython** → Python for .NET  
- **RubyPython** → Python embedded in Ruby  
- **Anaconda Python** → For Data Science / Big Data

## Python Features Recap
- Simple and easy to learn  
- Free and open-source  
- High-level programming  
- Platform-independent → Windows, Linux, macOS, Android  
- Portable → code runs across platforms without changes  
- Dynamically typed  
- Interpreted → Python Virtual Machine executes code line by line  
- Extensible → e.g., Jython  
- Embedded → devices like washing machines, TVs, microwaves  
- Extensive standard library  
- Huge community support

## Limitations
- Slower than compiled languages like C++  
- Not ideal for mobile apps (but **Kivy framework** allows cross-platform development)
